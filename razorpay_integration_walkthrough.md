# Razorpay Payment Integration Details

I have successfully integrated Razorpay payment gateway into your website. Below is a detailed breakdown of the changes made across the frontend, backend, and database.

## 1. Frontend Changes
The frontend now supports both **Cash on Delivery (COD)** and **Online Payments**.

- **External Script**: Added the Razorpay Checkout SDK to `client/index.html`.
  ```html
  <script src="https://checkout.razorpay.com/v1/checkout.js"></script>
  ```
- **Cart Page (`Cart.jsx`)**: 
  - Added a **Payment Method Selection** UI (COD vs Online).
  - Implemented the `handleCheckout` logic to trigger the Razorpay popup when "Online Payment" is selected.
  - Added a **Payment Verification** step that calls the backend after a successful transaction.
  - Improved the user experience with loading states during payment processing.

## 2. Backend Changes
The backend handles order creation and secure payment verification.

- **Dependencies**: Installed the `razorpay` package.
- **Routes (`orderRoutes.js`)**:
  - **New Route (`POST /api/orders/verify`)**: Verifies the Razorpay signature using your `RAZORPAY_KEY_SECRET` to ensure the payment is authentic.
  - **Updated Order Creation**: When an order is created with "Razorpay" as the method, the server automatically generates a Razorpay Order ID and sends it back to the frontend.
- **Environment Config**: Updated `.env.example` to include placeholders for your Razorpay API keys.

## 3. Database Changes
The `Order` model was updated to store payment-specific metadata.

- **File (`server/models/Order.js`)**: Added the following fields:
  - `razorpay_order_id`: Stores the unique ID generated by Razorpay.
  - `razorpay_payment_id`: Stores the actual payment transaction ID after success.
  - `razorpay_signature`: Stores the cryptographic signature for verification.
  - `paymentStatus`: Now tracks "Paid" vs "Pending" (Defaults to "Pending" for COD).

---

> [!IMPORTANT]
> **Action Required**: You need to add your actual Razorpay keys to your `.env` file in the `server` directory for the payment gateway to work:
> ```bash
> RAZORPAY_KEY_ID=your_key_id_here
> RAZORPAY_KEY_SECRET=your_key_secret_here
> ```
> *You can get these from your [Razorpay Dashboard](https://dashboard.razorpay.com/).*
